Following these steps to run the CREST-SP tracker on our dataset:
1.Download the CREST code from https://github.com/ybsong00/CREST-Release
2.Copy the .m files to the /CREST directory and replace the original ones. 
3.In the config_list.m, change the test_source (Line 7) to the directory where our dataset stored.
4.In the Demo.m, change the datasetPath (Line 31) to the directory where our dataset stored.
5.run Demo.m
