Following these steps to run the BACF-SP tracker on our dataset:
1.Download the BACF code from http://www.hamedkiani.com/uploads/5/1/8/8/51882963/bacf_toupload.zip 
2.Copy the .m files in this directory to the BACF directory, and replace the original .m with the ones in this directory.
3.In the BACF_Demo.m, change the base_path (Line 31) to the directory where our dataset stored.
4.run BACF_Demo.m
