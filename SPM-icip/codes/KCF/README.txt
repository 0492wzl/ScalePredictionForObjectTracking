Following these steps to run the KCF-SP tracker on our dataset:
1.Download the KCF code from http://www.robots.ox.ac.uk/~joao/circulant/tracker_release2.zip
2.Copy the .m in this directory to the KCF directory, and replace the original ones.
3.In the run.m file, change the data_path(Line 21) to the directory where you put our dataset in.
4.run run.m
