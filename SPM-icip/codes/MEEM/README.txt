Following these steps to run the MEEM-SP tracker on our dataset:
1.Download the MEEM code from http://cs-people.bu.edu/jmzhang/MEEM/down_code.php?Down=MEEM_v1.1_release.zip
2.Replace demo.m and MEEMTrack.m in the /MEEM directory with the ones in this directory.
3.Copy scale_estimation.m into the /MEEM directory.
4.In the demo.m file, change the img_path(Line 26) to the directory where you put our dataset in.
5.run demo.m
